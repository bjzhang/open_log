发表与 2005-10-11 21:31:25
在实际工作中，有时往往会因为项目进度要求需要用一款自己以前未曾使用过的MCU来进行产品的开发，而且进度要求很急，为此，笔者结合自己的经验，谈一下如何以最快的速度来掌握和面对此情况： 1。认真阅读Datasheet中的第一页（Summary 部分），一般该页都会将此器件的所有Features和主要参数进行详细地说明； 2。快速浏览一遍其CPU的结构和寻址方式； 3。快速浏览一遍其Instruction Set（仅仅Summary部分），这样在编程时会有一个初步的认识； 4。结合项目的设计要求和资源的使用情况，一边编程一边认真阅读相应的章节；这样，随着项目的完成，该芯片的相关功能、Instruction Set也就自然掌握了！

const变量的作用（自网上）：
const int a;	//a是一个常整型数
int const a;	//a是一个常整型数
const int* a;	//a是一个指向常整型数的指针(整数不可以修改，指针可以)

const意味着“只读”。提示你这个变量是只读的，函数的输入参数一般用const类型，例如字符串复制函数：
char *strcpy(char *dest,const char *src)
const char *src表示src指向的变量是常量，不可以修改，src的地址可以修改。

RFID, 国家标准, EPCGlobal标准
自: 电子工程专辑 工业控制网站
http://www.industrialcontrols.eetchina.com/ART_8800431232_2500002_80a2afd4200608.HTM?1000010163&8800431232&click_from=1000010163,8883567706,2006-08-25,ICCOL,EENEWS

EPC RFID渐入佳境，中国标准又要望洋兴叹？
上网时间: 2006年08月25日

根据最近的一份研究报告显示，随着EPCGlobal迅速推出标准，而这个标准又逐渐为中国本土制造商所采纳，中国RFID国家标准的成功机会日渐微薄。

ABI Research分析师Junmei He在这份报告中写到，“现阶段，RFID标准在全球推广采纳的步伐已经开始提速，而中国政府何时推出RFID国家标准还遥遥无期。”分析师认为，中国政府的沉默与EPCGlobal公司的积极推广让中国RFID标准的制定面临更多的不确定性。当官员们想建立中国的RFID标准的努力依然难产时，瞬息变换的情况成为接下来将要面临的挑战。

中国多年来一直在进行RFID标准的建设工作，业界消息来源最近预测，2006年秋天将推出一份草案，并希望能在2007年推出最终版本。这份非强制的官方标准将密切跟踪国际标准，据悉在理论上将比采纳EPCglobal的系统成本更低。而与此同时，EPCglobal率先抓住了主动权，从而抢占先机。在2006年4月，EPCglobal香港发布了EPCglobal产业支持计划（EISP），以给采用EPC的香港和华南地区的企业提供财政、技术和执行等方面的支持。

由于大多数中国的产品出口到海外，这些制造厂需要遵照客户要求使用国际标准。不久前，ISO/IEC批准了EPC Gen 2 Class 1 UHF标准，作为使用860-960MHz ISM器件的18000-6 RFID接口补充标准。中国通常是ISO标准的拥护者，但是尚未停止建立自己标准的努力。

中国仍然在进行自有编码系统的开发，称为国家产品代码(National Product Code)，这个编码将与EPCglobal的电子产品代码(EPC)竞争。中国也将建造和维护自己的数据库来存储关于产品、制造商和运输方法的信息，而非订购来自EPC的数据，后者的数据由美国Verisign公司持有。

分析师在报告中指出，现实情况是中国的制造商正在研究学习EPCGlobal系统，以便他们能够与沃尔玛(WalMart)等客户的供应链系统平滑集成。他说，EPCGlobal通过为制造商提供财务、技术和实施方面的支持，对制造商伸出“橄榄枝”，这些举措使EPC成为事实上的标准。他在报告中谈到，“现实上，对于中国政府来说，何时推出国家标准已经不再重要，这个标准已经迟到了。更紧急的问题是，政府如何与EPCglobal合作或者谈判；建立保护本国RFID产业发展的政策；同时，在未来的RFID时代如何保护中国标签使用者的权益。”

ABI指出中国政府如今要想建立自己的标准，至少面临的三大挑战：首先，中国制造商的供应链合作伙伴将要求他们采用符合国际标准的 RFID标签，即EPCglobal UHF G2；其次，由于中国制造商的外国客户正计划采用RFID，他们将会要求发运的货物采用支持EPCglobal UHF G2标准的RFID标签；最后，EPCglobal在中国制造业核心地区的积极扩张行动正使EPCglobal UHF G2标准日益变成为内地的业界标准。
