c语言, 给孙建辉的回信
注释问题: 不要使用汉语拼音做注释.
代码风格: 注意缩进. 好的编程风格便于代码的调试和维护, 也便于别人看懂代码.
例如下面的语句中"ADON = 0;", 像是属于else语句的.
       if (ADC_VALUE>205)  gpio4=1;
       else      gpio2=1;   
          ADON = 0; // ADC powered off
可以改为
	if ( ADC_VALUE > 205 )
		gpio4=1;
	else
		gpio2=1;
	ADON = 0; // ADC powered off

嵌入式, LEON, iMPACT, prom, ***
iMPACT中从bit文件生成"*.mcs"文件的步骤, 以GR-XC3S-1500开发板为例

自: LEON邮件列表
Trouble with iMPACT for GR-XC3S-1500 board
Re: Trouble with iMPACT for GR-XC3S-1500 board

1) I have synthesized my .vhd file.
2) I open iMPACT by double clicking "Generate PROM, ACE or JTAG File".
3) The boundary scan is detected and shows 2 PROMs XCF04s and XCF01s.
4) I choose "File Mode".
5) I choose "PROM Formatter"
6) I right click and Launch Wizard.
7) I manually add XCF04s in position 0 and XCF01s in position 1.
8) I choose my .bit file and choose to generate PROM files.
9) do not: "click on SVF-STAPL-XSVF. I right click each of the PROMs andassign the respective .mcs file."
do: return from 'File Mode' back to 'Configuration Mode'. You do this by selecting the Menu Option 'Mode' => 'Configuration Mode' on the top line. You want the 'Boundary Scan' tab to be active. 
Now right-click and select 'Initialise Chain'. If the JTAG cable is connected to the board and the board is powered 'on' then the boundary scan should detect the three devices "XCF04, XCF01 and XC3S1500" similar to figure 3-14 in the user manual:
http://www.pender.ch/docs/GR-XC3S-1500_user_manual.pdf
Impact will then ask you to assign the files. Assign:
xxxx_1.mcs file to the first configuration PROM (XCF04)
xxxx_2.mcs file to the first configuration PROM (XCF04)
xxxx.bit file to the FPGA (XC3S1500)
10) I then choose both the PROMs and choose Program with "load FPGA"
option checked.
11) The board jumper JP9 is in "101" mode.
If impact reports 'bit errors' when loading the FPGA bit file, see this
answer for help:
http://www.pender.ch/faq.shtml#faq16
The 'SVF-STAPL-XSVF' which you selected is a different BSCAN format and option. Please refer to the Xilinx Impact documentation for more information on how to use this.
